Задание 5. Верстка экрана персонажей

На этом этапе наша задача  - научиться работать со статическими ресурсами приложения (цветами, текстовыми стилями, картинками и т. д). Также будем тренироваться в верстке.

Одним из наиболее популярных инструментов для разработки мобильного дизайна является Figma. Необходимо научиться работать с ним. Для удобства вы можете использовать десктопную версию. Макет нашего приложения находится здесь.

Подготовка констант (постоянных значений)

  • Создаем папку lib/constants/

В неё мы будем помещать данные приложения, которые всегда будут неизменными, такие как цвета, текстовые стили, пути к ресурсам, и т. д.

  • В папке constants создаем файл app_colors.dart, который будет содержать класс AppColors() -  все цвета нашего приложения. На данном этапе достаточно вынести цвета с экрана персонажей (только светлая тема), который мы будем верстать:

Внимание: нижнее меню (BottomNavigationBar) пока создавать не нужно, его мы добавим в одном из следующих заданий.

Это один и тот же экран, в котором два режима отображения, которые можно переключать (к этому перейдем позже).  Цвета каждого элемента вы можете посмотреть в Figma. Для этого выберите необходимый элемент -> в меню справа выберите раздел Inspect -> найдите параметр Colors -> скопируйте значение цвета в формате Hex.

Далее для каждого из используемых нами цветов мы создадим константу, которой присвоим объект Color(0xFF#наше скопированное значение#).

Примечание: FF - это прозрачность, устанавливаем непрозрачный цвет (FF равняется единице в шестнадцатеричной системе счисления).

  • В папке constants создаем файл app_styles.dart, который будет содержать класс AppStyles() - все текстовые стили нашего приложения, чтобы не создавать их снова и снова и не засорять код. На данном этапе достаточно вынести стили с экрана персонажей, который мы будем верстать. По аналогии с цветами мы копируем параметры стиля (кегль шрифта, толщину и т.п.) из имеющихся данных и создаем константы, описывающие шрифты в классе AppStyles:

Вёрстка экрана со списком персонажей

Итак, подготовительная работа закончена. Теперь нужно сверстать экран персонажей.

  • Иконки можно использовать следующие: Icons.search, Icons.filter_list_alt, Icons.grid_view.
  • Добавить переключение между списком и сеткой по тапу на кнопке переключения. Помните, что обновление экрана можно вызвать через метод setState.
  • Цвет строки статуса телефона можно задать с помощью SystemChrome.setSystemUIOverlayStyle

Напоминание

Не забывайте о хорошей читаемости вашего кода, расставляйте запятые перед скобками (чтобы автоформатирование работало корректно), давайте классам и переменным понятные названиях, разбивайте слишком длинные файлы на части. Желательно, чтобы  в одном файле было не более 200 строк.
